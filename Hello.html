H-Hyper : Being Super Interactive (HTML is very interactive as it can executed in any browser, and can get the DATA from anywhere in the web)
    Some features of HTML which makes it hyper:
  • Anchor tag "a"
  • Forms
T-text: in HTML Text means any visible component in a web page. These reside in the body of page. Text can be plain text, images, video, links, tables, etc. M-markup: in HTML the tags and styling are major part that describes the metadata of the page with the help of tags and css L-Language: all computer languages have common features of having:
  • Variables: placeholder to actual value
  • Statements: Lines of codes, like Statements
  • Control Structure: if else if...else, switch case
  • Loops: Repetitive structures
  • Function: group of statements solving a particular problem
In HTML we have JavaScript handling the language part.

CSS: Cascading: following HTML file Style: style the HTML content Sheet: the place where we write the style code

JS: JavaScript is the language behind HTML.

TS: Type script is the new language which is strongly typed TS gets compiled into javascript and then executed in HTML.
